# Installation

1. Install rbenv https://github.com/rbenv/rbenv
2. Install ruby 2.6.5 via rbenv
```
rbenv install 2.6.5
```
3. Install bundler
```
gem install bundler
```
4. Bundle install
```
bundler install
```
5. Run middleman server
```
middleman server
```
6. Site will live on http://localhost:4567
